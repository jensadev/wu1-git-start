Arbetsflöde/gång

  1. Jens pratade om HTML, det var Tim Berners-Lee fel
  2. HTML är ett märkspråk
    • HTML består av element
    • element skrivs med taggar
    • element kan ha attribut
  3. Skapa en fork av ett repository
  4. Klonade repot med github desktop (gröna knappen)
  5. Öppnade repot i Visual studio code
  6. Öppnade filen index.html
  7. Ändrade i index.html, skapade listor och bild
  8. För att visa bilden så gjorde vi en images mapp och sparade bilden i den
  9. Sedan använde vi img taggen med src attributet för att få bilden att laddas
  10. Bilden var gigantisk, vi öppnade den i paint och minskade storleken
  11. Vi gjorde en fin commit med ett begripligt meddelande
  12. Vi pushade allt till Github
  13. Vi slog på github pages för att få en webhost med allt
  14. Vi laddade även in en google font
  15. Adda fonten med pluset och lägg sedan till länken i head taggen
  16. Sist så lägger vi till font-family i css filen
  17. För att spara denna lista på github, behöver jag nu göra en commit och sedan push
  18. profit $$$$
